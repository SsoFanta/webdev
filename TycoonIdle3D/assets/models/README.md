# Model Slots

Drop your high-quality models here to replace fallback meshes.

Use these exact filenames:

- `mine.glb`
- `smelter.glb`
- `fabricator.glb`
- `power.glb`
- `trade_hub.glb`

Notes:

- Keep the model base near `y = 0`.
- Keep orientation consistent (forward +Z preferred).
- Keep scale close to meters (`1 unit = 1 meter`).
